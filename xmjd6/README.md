#### 功能有：

#### 630			630个与原词库不冲突的简词（原生自带）

#### 以下为拓展功能：

#### 			流式			可以打句子，有词汇自动调频

#### 			A键			自造词

#### U键    		开启全拼反查功能<拼音查全码> 

#### V键    		开启二分反查功能<拼音拆分文字可查全码>

#### 			O键			超级繁体(词库的扩展，依旧是键道规则)

#### 		注意：默认为顶功开启，流式关闭

#### 					想使用流式请查看xmjd6.schema.yaml
