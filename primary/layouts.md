# 键位阐述

星空键道采用了5个字母对应5个笔画，分别是：

| **A** | **乛（横折、横竖）** |
| :---: | :------------------: |
| **V** |     **一（横）**     |
| **U** |     **丿（撇）**     |
| **I** |     **丨（竖）**     |
| **O** |   **丶（点、捺）**   |

我们在全拼当中使用的**「A - Z」**  一共26个大写字母不变，作为：**声母**
