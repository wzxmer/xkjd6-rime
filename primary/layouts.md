# 笔画键位

以21个键的双拼布局为基础，星空键道采用了5个键对应5个笔画，分别是：

| **A** | **乛（横折、横竖等）** |
| :---: | :--------------------: |
| **V** |      **一（横）**      |
| **U** |      **丿（撇）**      |
| **I** |      **丨（竖）**      |
| **O** |    **丶（点、捺）**    |
